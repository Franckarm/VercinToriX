# VERCINTORIX

**Langage de programmation universel** — logique trinaire native,
cube chromatique comme espace mémoire, ADN quadrinaire.

## Philosophie

> Penser comme le vivant. Parler comme l'humain.
> Voir comme la lumière. Se lier comme la matière.

## Caractéristiques uniques

- **Cube chromatique** — espace mémoire 3D où position = couleur = adresse ADN
- **Logique trinaire** — CERTAIN / PROBABLE / BRUME — l'incertitude native
- **ADN quadrinaire** — encodage A/G/C/T avec 4 états, 16M couleurs en f32
- **DUBI contextuel** — décision qui parle le vocabulaire du domaine
- **HELIX** — comportements variants et évolutifs avec allèles
- **GWEL / ADBERT** — miroir et symétrie dans l'espace chromatique
- **PAR** — parallélisme structurel natif
- Vocabulaire **gaulois** (structures) + **français** (actions)

## Exemple

```vercintorix
TARVOS AUTO précision(f32) contexte(hardware)

NEMETON Bonjour {
    déclarer message → "Bonjour VERCINTORIX"
    afficher message → écran
}
```

## Lancer un programme

```bash
node vercintorix.js programme.vtx      ← interpréteur JS
cargo run -- programme.vtx             ← noyau Rust
```

## Structure du projet

- `spec/` — spécifications officielles V0.2 et V0.3
- `interpreteur/` — interpréteur JavaScript
- `noyau_rust/` — noyau Rust en cours de développement
- `exemples/` — programmes de démonstration

## Version

- **V0.2** — Interpréteur JS fonctionnel, toutes les primitives de base
- **V0.3** — ADN quadrinaire, DUBI contextuel, HELIX multi-allèles,
             GWEL/ADBERT/PAR, cube chromatique complet
